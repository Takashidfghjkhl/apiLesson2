# Конвертер валют

конвертирование валют с помощью API

## Как установить

1. cкачать aрхив с репозиторием
2. получить `api` ключ на [сайте](https://www.exchangerate-api.com/docs/standard-requests)
3. создать в папке файл `.env` и записать в переменную `APIKEY` свой ключ

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как Запустить проекта

1. открыть папку с файлами, например, через VSC
2. открыть терминал и прописать:
```
python main.py --base (код базовой валюты) --target (код целевой валюты) --amount (сумма для конвертирования)
```
